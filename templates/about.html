About This Project

Jack Coyne created this project as part of the coursework for the Digital Product Management class. The aim is to demonstrate how artificial intelligence can be employed to predict salaries based on various inputs such as age, experience, and education level. This application serves as a practical example of integrating machine learning models with a front-end interface to provide real-time predictions.

This project integrates technologies including Python, Flask, and Bootstrap to create a responsive web application. It is designed to be informative and user-friendly, allowing users to understand how different factors might affect their potential earnings in the tech industry.
